# Skill Portfolio

データサイエンスの視点と実装力を見せるポートフォリオサイト。
分析から価値あるアウトプットまでのプロセスを言葉と制作物で伝えます。

## What to See

このポートフォリオでは、単なる制作物一覧ではなく、以下の流れを見せることを重視しています。

- どんな視点で課題を見ているか
- 学習や制作をどう進めているか
- 分析や実装をどのように価値へつなげたいか

## Pages

- **Home** - ポートフォリオの全体像とSkill紹介
- **About** - 人物像、興味分野、学習姿勢
- **Portfolio** - 制作物とその背景
- **Learning** - 現在の学習状況と進行中のタスク

## Design

- フローベースのビジュアル設計
- 主要ページへの軽い導線設計
- レスポンシブ対応
- アニメーション演出

## Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- Google Fonts (Manrope, Noto Sans JP)
- CSS Grid / Flexbox

## Features

- Perspective セクション：幾何学的な情報フロー表現
- Portfolio Page：制作物を検索しながら確認できる一覧
- Learning Page：タスクを追加・削除できる学習ログボード（LocalStorage利用）
- Smooth Animations：Cubic-bezier easing

## Getting Started

1. ファイルをクローン
2. `index.html` をブラウザで開く

## Notes

- 現在は静的サイトです。
- 外部APIやバックエンドは使っていません。
- `.env` や秘密情報を使う前提の機能はありません。
