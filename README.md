# Skill Portfolio

データサイエンスの視点と実装力を見せるポートフォリオサイト。
分析から価値あるアウトプットまでのプロセスを言葉と制作物で伝えます。

## 📄 Pages

- **Home** - ポートフォリオの全体像とSkill紹介
- **About** - 人物像、興味分野、学習姿勢
- **Portfolio** - 制作物とその背景
- **Learning** - 現在の学習状況と進行中のタスク

## 🎨 Design

- フローベースのビジュアル設計
- インタラクティブなタブUI
- レスポンシブ対応
- アニメーション演出

## 🛠️ Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- Google Fonts (Manrope, Noto Sans JP)
- CSS Grid / Flexbox

## 📝 Features

- Perspective セクション：幾何学的な情報フロー表現
- Selector Tabs：複数ページの内容をインタラクティブに比較
- Learning Page：タスク管理機能（LocalStorage利用）
- Smooth Animations：Cubic-bezier easing

## 🚀 Getting Started

1. ファイルをクローン
2. `index.html` をブラウザで開く
